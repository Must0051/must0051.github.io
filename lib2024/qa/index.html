  10. Qus 10: What is the use of name and value attributes? Give two examples

    Ans: the use of name is put the name into HTML button or form and the value attribute saves the input and submits it to the server. For two examples: 1. <button name="subject" type="submit" value="name">Name</button> 2. <button name="subject" type="submit" value="value">Value</button>

  11. Qus 11: Give an example of inline style.

  12. Qus 12: How would you link an external style sheet? Give example.

    Ans: Use link tag after head tag at the beginning for an external style sheet for example: <link rel="stylesheet" type="text/css" href="style.css">

  13. Qus 13: Create CSS block to center an element both horizontally and vertically that has position:absolute and has width:200px and height:200px.

    Ans: .center {
    position: absolute;
    width: 200px;
    height: 200px;
    padding: 20px;
    text-align: center;
    }

  14. Qus 14: Create CSS keyframes block called "spin" that rotates an element.

    Ans: @keyframes spin {
    from {
    transform: rotate(0deg);
    }
    to {
    transform: rotate(360deg);
    }

    }
    @keyframes spin {
    from {
    transform: rotate(0deg);
